# Roland MIDI test corpus

Sample `.mid` files from a Roland SC-55 / DOS GM game MIDI collection (John Novak comparison rips and classic era game arrangements).

Used by `src/lib/roland-corpus.test.ts` for import + export validation smoke tests.

| File | Source game |
|------|-------------|
| `arena-dungeon1.mid` | The Elder Scrolls: Arena |
| `discworld-alley.mid` | Discworld |
| `space-quest-v-starcon-academy.mid` | Space Quest V |

Replace or extend this folder anytime — tests only depend on these three filenames.
